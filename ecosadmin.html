How this works

The chatbot saves its transcripts to this browser's local storage — same as the rest of your site data. That means you'll see chats that happened in this browser (handy for testing), but conversations visitors have on their own devices won't show up here automatically, since this site has no backend/database. If you want a true cross-visitor chat log, that would need a small server to collect messages centrally — happy to help set that up if you want it.

Important — how publishing works

Everything you edit here is saved to this browser's local storage only. It is not saved to your live site automatically — visitors (and you, on any other device or browser) will only ever see whatever was baked into index.html the last time you exported it. After making changes, click "Download updated index.html" below, then replace index.html in your GitHub repo with the downloaded file and push — that's what actually updates the published site.
